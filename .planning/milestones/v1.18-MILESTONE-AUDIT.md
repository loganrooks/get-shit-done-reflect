## Tech Debt Summary

### Phase 54: Stale Installed CI Hooks
The Phase 54 CI cache scoping fix (per-project `gsd-ci-status--{repo}--{branch}.json`) exists in source `hooks/dist/` but has not been propagated to the installed `.claude/hooks/` copies. The installed writer and reader agree on the old unscoped filename, so CI display works within this install. **Fix:** `node bin/install.js --local`.

### Phase 47: sensors.cjs Discovery Pattern
`sensors.cjs` uses `gsd-*-sensor.md` regex to discover sensor files, but installed agents use `gsdr-*-sensor.md` prefix. The `sensors list` CLI diagnostic command returns zero sensors in production installs. **Non-blocking:** the `collect-signals` workflow uses its own bash glob with correct namespace and does not call `sensors list`.

### General Items
- **$HOME path double-slash** in local install paths (`$HOME/./.claude/`). Shell resolves correctly. Cosmetic only.
- **Flaky automation test** (Phase 48 ETIMEDOUT on process spawn under load). Environmental, not a code regression.
- **renameSync crash test limitation** (Phase 50). Test environment can't mock cross-tmpdir renameSync. Documented as design decision, original data safety verified.
