**GATE-02 step (example; adjust to match existing CI structure):**

```yaml
- name: GATE-02 merge-strategy conformance
  run: |
    set -e

    # 02a: no `gh pr merge` without `--merge`
    if grep -rn "gh pr merge" agents/ get-shit-done/ commands/ .claude/ .codex/skills/ skills/ 2>/dev/null \
        --include='*.md' \
        | grep -v "\-\-merge" \
        | grep -v '# historical'; then
      echo "::notice title=GATE-02::gate_fired=GATE-02 result=block reason=gh_pr_merge_without_merge_flag"
      echo "::error::GATE-02: found `gh pr merge` invocation without `--merge` flag"
      exit 1
    fi

    # 02b: no `git merge --squash` (outside historical comments)
    if grep -rn "git merge --squash" get-shit-done/ agents/ commands/ skills/ .codex/skills/ 2>/dev/null \
        --include='*.md' \
        | grep -v '# historical'; then
      echo "::notice title=GATE-02::gate_fired=GATE-02 result=block reason=git_merge_squash_found"
      echo "::error::GATE-02: found `git merge --squash` invocation"
      exit 1
    fi

    echo "::notice title=GATE-02::gate_fired=GATE-02 result=pass"
```

**GATE-13 step:**

```yaml
- name: GATE-13 dispatch-contract restatement
  run: |
    set -e

    # 13a: no `Task(...)` blocks with `{...}` template expression at the model= field
    # This uses a multi-line regex: find `Task(` then the next 5 lines, look for a model= line with `{x}` binding
    MATCHES=$(grep -rn -A 5 'Task(' agents/ get-shit-done/workflows/ commands/ --include='*.md' 2>/dev/null \
        | grep -E 'model\s*=\s*"\{[^}]+\}"' \
        || true)

    if [ -n "$MATCHES" ]; then
      echo "::notice title=GATE-13::gate_fired=GATE-13 result=block reason=template_variable_in_dispatch_contract"
      echo "::error::GATE-13: found Task() dispatch with unresolved template variable at model= (compaction-unsafe)"
      echo "$MATCHES"
      exit 1
    fi

    echo "::notice title=GATE-13::gate_fired=GATE-13 result=pass"
```

Option B (alternative): make the GATE-13 step a warning (not failure) until Plan 12 has landed; convert to failure in Plan 12's closing step. This is weaker per DC-1 (would make the gate advisory temporarily) so Option A is preferred.
