  <action>
Create `get-shit-done/bin/lib/archive.cjs` exporting `archiveAgentOutput(options)`.

**Function contract:**

```javascript
// archiveAgentOutput({ sessionId, reason, paths, phaseNumber, dryRun })
// Returns: { archived: [archive_path...], missing: [...], errors: [...] }
archiveAgentOutput(options)
```

**Algorithm:**

1. Determine archive root: `.planning/phases/${phaseNumber}-*/.archive/`. If `phaseNumber` not provided or phase directory not found, fall back to `.planning/archive/` at repo root.

2. Compute timestamped subdirectory: `${YYYYMMDDHHMMSS}-${sessionId}-${sanitize(reason)}/`.

3. For each path in `paths[]`:
   - Verify path exists; if missing, record in `missing[]` and continue (not an error — path may have been deleted before archive).
   - Compute `src → archive_path = archive_root/<subdir>/<basename>`.
   - `mkdir -p` the subdir.
   - `mv src archive_path` (use `fs.renameSync` for same-filesystem; fall back to copy+rm for cross-filesystem).
   - On success, record in `archived[]`.

4. Emit fire-event: `console.log("::notice title=GATE-12::gate_fired=GATE-12 result=archived path=" + subdir + " reason=" + reason)`.

5. Do NOT write a README or metadata file into the subdir — keep the archive transparent. If metadata is needed, callers can pass a `--metadata <json>` flag and the library writes `.meta.json` inside the subdir.

**CLI:** `gsd-tools agent archive --session-id <id> --reason <reason> [--phase <N>] [--dry-run] --paths <path> [<path>...]`.

**Register in `gsd-tools.cjs`:** add `agent archive` route + help text. The `agent` subcommand namespace may be new; check existing commands first and follow the `phase`, `state`, `kb`, `handoff` pattern.

**Idempotency:** invoking twice with same session+reason creates a new timestamped subdir (safe; evidence accumulates). Calling with paths that don't exist: no error, records missing.

```bash
# GATE-12: archive partial output before redispatch
if [ -f "$OUTPUT_PATH" ]; then
  node $HOME/.claude/get-shit-done-reflect/bin/gsd-tools.cjs agent archive \
```
