Create `get-shit-done/bin/lib/handoff.cjs` exporting `resolveHandoff(options)` that implements GATE-04a + GATE-04b.

**Function contract:**

```javascript
// Returns { action: 'loaded'|'archived'|'hard_stop', archive_path?: string, reason?: string }
// Exits non-zero via caller on hard_stop
resolveHandoff({ continue_path = '.continue-here', state_path = '.planning/STATE.md', auto = false })
```

**Steps (GATE-04b first):**

1. If `continue_path` does not exist: `{ action: 'loaded', reason: 'no_handoff_present' }` â€” no-op, not an error.

2. Read file `mtime`; read STATE.md `last_updated` YAML frontmatter field; compute `git log -1 --format=%ct -- .planning/STATE.md` for most recent commit touching STATE.md.

3. **Staleness check** (GATE-04b): if ANY of:
   - `mtime(continue)` < parse(`STATE.last_updated`)
   - `git log --since=$mtime -- .planning/STATE.md` returns â‰Ą 1 commit on `main` after the file was written
   - A `session_id` embedded in the handoff YAML matches a session_id already recorded in STATE.md's decisions or activity history (grep `session_id: <id>` in STATE.md)
   â†’ `{ action: 'hard_stop', reason: 'stale'|'newer_state'|'duplicate_session' }`.
   Emit stdout `::notice::gate_fired=GATE-04b result=hard_stop reason=<reason>`.
   Exit non-zero (exit code 3 â€” distinct from other exit codes).

4. **Archive-and-load** (GATE-04a): if not stale:
   - Read handoff content (caller will parse).
   - Compute archive path: `.planning/handoff/archive/$(date -u +%Y%m%dT%H%M%S)-${session_id:-unknown}.continue-here.md`
   - `mkdir -p .planning/handoff/archive/` (create if missing).
   - `mv continue_path archive_path`.
   - Emit `::notice::gate_fired=GATE-04a result=archived path=<archive_path>`.
   - Return `{ action: 'archived', archive_path, content: <string> }`.

**CLI registration:** `gsd-tools handoff resolve [--continue-path PATH] [--state-path PATH] [--auto]`. Exit codes:
- 0: loaded or no-op (caller consumes content)
- 3: hard-stop staleness (caller aborts)
- other: error
