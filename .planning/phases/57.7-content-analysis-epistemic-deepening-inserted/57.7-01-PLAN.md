Run Spike C3 end-to-end to answer Q1 (Is a usable tokenizer available for Claude model family under the stipulated 15%/20% thresholds?). This plan owns DESIGN.md → BUILD → RUN → DECISION.md entirely within the phase — no upstream spike-programme infrastructure dependency (follows 57.5 E5.8 in-phase precedent).

If DECISION.md picks `approve_top_level_dependency`, Plan 08 may add the dependency; otherwise Plan 08 ships schema-only regardless of tokenizer PASS verdict.

Output: DESIGN.md (plan + acceptance thresholds), DECISION.md (chosen tokenizer OR schema-only verdict + production_dependency_decision), supporting-data (per-session deltas, evaluation corpus metadata).
