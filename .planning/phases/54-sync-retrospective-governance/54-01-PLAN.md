```javascript

try {
  repoName = execSync('git remote get-url origin', { encoding: 'utf8', timeout: 5000 })
    .trim().split('/').pop().replace(/\\.git$/, '');
} catch {
  try { repoName = require('path').basename(process.cwd()); } catch {}
}
```

2. Change the cache filename from the hardcoded `cacheFile` (passed via JSON.stringify from the parent) to a scoped version. The parent process (lines 10-11) currently computes `cacheFile` and passes it via `JSON.stringify(cacheFile)`. Change the parent's cacheFile to just the cacheDir, and let the child compute the full scoped filename:
   - Parent (line 11): Change `const cacheFile = path.join(cacheDir, 'gsd-ci-status.json');` to pass only `cacheDir` into the child.
   - Child: After deriving `repoName` and `branch`, compute: `const cacheFile = path.join(cacheDir, 'gsd-ci-status--' + repoName + '--' + branch + '.json');`
   - Update the `JSON.stringify(cacheFile)` reference in the spawn to `JSON.stringify(cacheDir)` and rename the variable in the child accordingly.

3. Use readable `--` delimiters (not hashes) per research recommendation. Format: `gsd-ci-status--{repoName}--{branch}.json`.
