# Governance Recommendations

This directory holds recommendation artifacts that sit between reviewed
deliberation/research work and direct roadmap or workflow modification.

These artifacts are not themselves open deliberations, and they are not yet
adopted roadmap decisions. They are governance-facing recommendation memos used
to route reviewed insight into later planning changes with clearer traceability.
