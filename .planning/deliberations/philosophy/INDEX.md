# Philosophical Deliberations Index

**Created:** 2026-03-03
**Purpose:** Epistemological foundations for GSD Reflect's self-improvement system. These documents are citable in requirements, design decisions, and signal interpretation via the `philosophy:` motivation type.

## Citation Format

In requirements and design documents, cite as:
```
*Motivation:* `philosophy: <framework>/<principle>` — <explanation>
```

Example:
```
*Motivation:* `philosophy: falsificationism/severe-tests` — passive verification-by-absence is a weak test
```
