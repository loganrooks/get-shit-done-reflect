## Problem

The current corpus grounding for the framing-critique / structural-norms
deliberations and the related review set is usable but still provisional. The
user explicitly judged that we do not yet have the proper paper set for at
least the new framing-critique deliberation, and likely for some of the related
reviews as well. The present citations should therefore be treated as
insufficient rather than authoritative.

The user also expressed hesitation about whether this should be formalized as a
todo in the system at all. That hesitation matters: this capture is meant as a
lightweight reminder for later reconsideration, not as proof that the current
grounding is invalid or that a roadmap task is already warranted.

The practical dependency is `arxiv-sanity-mcp`: once it is up and running
properly, we should be able to revisit the literature set with a better search,
selection, and comparative review process.

## Solution

After `arxiv-sanity-mcp` is operational enough to support stronger literature
exploration:

1. Revisit the paper set for:
   - metaphor-awareness / framing-critique deliberation
   - structural norms / practical judgment deliberation
   - the five corpus-grounded deliberation reviews
2. Reassess whether the currently cited papers are:
   - the right papers
   - cited for the right claim domains
   - still too thin, adjacent, or weakly transferable
3. Keep insufficiency explicit in the artifacts unless and until a stronger
   paper set is established.
4. Reconsider whether any of this should remain as lightweight review notes
   rather than formalized framework method.
